О проекте
LLM Attack Simulation

Это веб-приложение предоставляет полный интерфейс для симуляции и анализа атак через языковые модели. Система включает в себя генератор фишинговых атак, детектор угроз, симулятор сети и комплексную аналитику.

Проект разработан для исследователей, специалистов по безопасности и образовательных учреждений, которые изучают методы защиты от современных угроз.

Технологии:
  • Python 3.8+
  • HTML5 + CSS3
  • JavaScript
  • Bootstrap 5
  • Flask
